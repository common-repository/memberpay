A Memberpress payment gateway integration with Midtrans for membership subscriptions.

== Description ==

MemberPay is the best tool to make payments for your Memberpress subscription via Midtrans.

You can easily connect with payment methods that apply in Indonesia.

== Plugin Installation and Demo ==

== Plugin Features ==

= Payment Link =

Our plugin will generate a payment link automatically. So your subscription becomes easy to pay.

= Auto Redirect =

After the subscription has been successfully created, our system will automatically take you to the payment page.

== Installation ==

= Minimum Requirements =

* MemberPress 1.9.51 or greater is recommended
* MemberPress Developer Tools 1.2.14 or greater is recommended
* MemberPress Courses 1.2.2 or greater is recommended
* PHP 8.1.11 or greater is recommended
* MySQL 5.7 or greater is recommended

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of MemberPay, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “MemberPay”, then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the MemberPay plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
